wasco is a python-wrapper for the wasco IODA-PCI12K4EXTENDED PCI
multifunction card. We use this card to control three sets of Osram
fluorescent tubes in colors red (color 60), green (color 66), and blue
(color 67).

For license see LICENSE.txt.

This module should run with python 2.5-2.7 or with python 2.4 and ctypes.

At the moment it runs only under MS Windows.
